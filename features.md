# Features Reference

## Core platform
- Grant discovery with 100+ sources (RSS + APIs), dedupe, and ingestion run logging
- Scheduled ingestion via InsForge Edge Function (cron) + manual refresh
- Personalized grant matching using profile embeddings and `match_grants` RPC

## Project workspace
- Projects tied to grants with status tracking
- Collaborative document editor with realtime sync, presence, and cursors
- Tasks and milestones for submission readiness
- Artifacts upload/download using a private storage bucket
- Submission package export (ZIP)

## AI capabilities
- Grant Scout ingestion and enrichment
- Compliance check (eligibility alignment)
- NIH-style review simulation
- Co-pilot drafting and editing assistance

## Collaboration
- Team invites with tokenized accept/decline flow
- Email invites via Resend
- Notifications on invite actions

## Profile intelligence
- Detailed research profile capture (career stage, focus areas, preferred funders)
- External profile links (faculty page, lab site, Google Scholar, ORCID, CV)
- AI profile analysis to extract expertise, tags, and grant-fit signals
