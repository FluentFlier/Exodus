# Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS, Shadcn/UI, Lucide React
- **Backend**: InsForge (PostgreSQL, Auth, Storage, Edge Functions)
- **Database**: PostgreSQL (with `pgvector` for AI)
- **Real-time**: InsForge Realtime + Y.js (for CRDT collaboration)
- **Rich Text Editor**: Tiptap
- **AI**: OpenAI (via InsForge)
- **Deployment**: InsForge Hosting / Vercel (TBD)
- **Version Control**: GitHub
- **Code Review**: CodeRabbit
